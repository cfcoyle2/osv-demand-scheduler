# Kabal Website Package

This folder is the Library copy of the Kabal HTML site.

## Current Entry File

- `index.html`

## Folder Use

- `assets/`
  - Images, icons, stylesheets, or scripts if the site is later split out of the single-file format.
- `docs/`
  - Project notes, source summaries, requirements, and change logs.
- `data/`
  - Static JSON, CSV, or reference data files used by the site.
- `archive/`
  - Old versions, exports, or retired supporting files.

## Notes

- The current site is self-contained in `index.html`.
- `SOURCE_README.md` remains at the project root as the original source summary copied from the Kabal project folder.
- If this site grows, keep the live entry point as `index.html` and move supporting files into the folders above.
