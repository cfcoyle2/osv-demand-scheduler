# Project Work Package Template

Use this template for business work that combines analysis, data, scripts, and presentation outputs.

## Structure

- `01_Briefs_and_Notes`
  - Working notes, manager briefs, assumptions, summaries.
- `02_Data`
  - Excel files, exported data, raw inputs.
- `03_Presentation_Source`
  - Slide outlines, markdown source, visuals prepared for decks.
- `04_Final_Deliverables`
  - Final PowerPoint, PDF, memo, or packaged outputs.
- `05_Scripts`
  - Automation scripts tied to the project.

## How to Use

1. Copy this folder into the correct library category.
2. Rename it to the project name.
3. Put source files in `02_Data`.
4. Draft written analysis in `01_Briefs_and_Notes`.
5. Build slide source in `03_Presentation_Source`.
6. Store final outputs in `04_Final_Deliverables`.
7. Keep any project-specific scripts in `05_Scripts`.

## Good Fit

- Logistics studies
- Decision support packages
- Leadership presentations
- Cost and schedule analysis
- Cross-functional working files
