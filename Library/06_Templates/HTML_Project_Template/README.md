# HTML Project Template

Use this folder as the starting point for a new HTML-based site or dashboard.

## Structure

- `index.html`
  - Main entry page.
- `assets/css/`
  - Stylesheets.
- `assets/js/`
  - JavaScript files.
- `assets/images/`
  - Images, icons, screenshots.
- `docs/`
  - Notes, requirements, source summaries.
- `data/`
  - JSON, CSV, exports, or static data files.

## Suggested Setup

1. Copy this folder into `01_Websites`.
2. Rename it for the project.
3. Update `index.html` title and content.
4. Add project notes in `docs/`.
5. Store any supporting data in `data/`.

## Good Fit

- Single-page HTML tools
- Internal dashboards
- Presentation-style HTML pages
- Reference guides

If a project stays fully self-contained in one HTML file, you can still use this template and keep the extra folders only when needed.
